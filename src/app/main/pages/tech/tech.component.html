Tecnologies


AngularJS

AngularJS is a JavaScript framework that simplifies the development of dynamic web applications through its declarative syntax and two-way data binding.

Flutter

Flutter is a cross-platform framework for building mobile apps with a single codebase, enabling developers to create beautiful and performant applications using a reactive UI toolkit.

HTML5

HTML5 is the latest version of the HTML language, providing enhanced multimedia support, semantic elements, and improved accessibility, enabling developers to create modern and interactive web pages and applications.

CSS3

CSS3 is the latest version of the CSS language, introducing new styling capabilities such as animations, gradients, and transitions, empowering developers to create visually appealing and engaging designs for web pages and applications.

Dart

Dart is a programming language developed by Google, known for its simplicity and efficiency in building cross-platform applications, including web, mobile, and desktop, with a focus on performance and scalability.
